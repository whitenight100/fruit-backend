1. git clone repo
'git clone https://github.com/whitenight100/fruit-backend.git'

2. create the MySQL database
dbname => 'fruity'
username => 'root'
password => ''

3. Run yii server with command in root directory

'php yii serve'

4. Run this command to insert the fruits into the MySQL DB

'php yii fruit'
